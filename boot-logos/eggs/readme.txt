Bootlogo Changer for TRIMUI/PowKiddy A66

usage: Launch trimui.sh or welcome.sh from GMenuNX Explorer.

trimui.sh and welcome.sh are exactly the same file.
This script writes .bmp file with the same name as itself as bootlogo.
So, if filename is logo.sh, logo.bmp will be written.

If you want to make a custom logo, make sure the bmp format is same and filesize does not exceed 512K.
- 320x240 16bit R5 G6 B5 < 524,288 bytes
